# ASP.NET作业：
为自己的家乡设计一个网站：
要求：
1.网站主要是展示和介绍自己的家乡；
2.要求网站整体风格一致；
3.要求展示家乡的历史、人文风情、自然景观（旅游景点等）、特色美食（美食故事及图片展示）、自己就读高中。
4.以上每个展示内容为一个网页，自己设计一个主页，从主页可以链接到以上每个页面。
